# Worldcup Index Guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以世界杯赛事相关的索引与导航内容为主，旨在提供结构化的信息展示与快速跳转功能。

## 项目简介

本项目是一个静态页面集合，所有页面均为独立的 HTML 文件，不依赖任何后端服务或数据库。页面内容聚焦于世界杯赛事相关的索引、指南与信息整理，适合作为轻量级的参考工具或信息归档库。

## 目录说明

```
worldcup-index-guide/
├── index.html           # 入口页面，提供总体导航
├── pages/               # 存放各独立 HTML 页面
│   ├── match-guide.html
│   ├── team-index.html
│   └── ...
├── assets/              # 静态资源（CSS、图片等）
│   ├── styles/
│   └── images/
└── README.md            # 本文件
```

- `pages/` 目录下存放所有独立页面，每个文件对应一个主题或功能模块。
- `assets/` 目录存放页面所需的样式表与图片等资源文件。

## 页面归档说明

- 所有页面均为纯 HTML，兼容主流现代浏览器。
- 页面之间通过超链接相互跳转，形成完整的导航结构。
- 内容仅用于信息展示与索引，不包含任何动态交互或用户数据收集功能。
- 归档内容会不定期更新，以保持信息的准确性与完整性。

## 维护说明

- 本仓库由维护者手动更新，不接受自动化提交或外部 Pull Request。
- 如发现页面链接失效或内容错误，可通过 Issues 反馈，但无法保证及时修复。
- 所有页面内容仅代表归档时的状态，后续可能进行调整或删除。

本项目仅供学习与参考使用。页面内容版权归原始信息提供方所有，本仓库仅作整理与归档。
